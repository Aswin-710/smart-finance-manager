# Smart Finance Manager

A full-stack fintech web application built using Angular, Node.js, and MongoDB.

## Features
- User authentication with JWT
- Expense and income tracking
- Loan and EMI management
- Real-time currency and stock data integration
- Analytics dashboard

## Tech Stack
- Angular
- Node.js
- Express.js
- MongoDB

## Project Structure
- frontend/ - Angular application
- backend/ - Node.js REST API
